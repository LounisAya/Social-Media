

# React Social Media App

## escrition
A simple social media app build with React , inspired by Facebook

## Setup instructions
- Clone the repo
- Run 'npm install'
-Run 'yarn start' to lunch the app

## Features 
- post feed with images, captions, likes, and comments
- profile page
- like and comment
- infinit scroll
